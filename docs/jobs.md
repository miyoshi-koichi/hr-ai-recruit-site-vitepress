# 募集職種

現在、以下の職種で採用を行っています。

## エンジニア職

### Webエンジニア

- **仕事内容**: 自社サービスの開発・運用
- **必須スキル**: HTML/CSS/JavaScript、いずれかのサーバーサイド言語
- **歓迎スキル**: React、Vue.js、TypeScript

### ゲームエンジニア

- **仕事内容**: スマホゲームの開発
- **必須スキル**: Unity または Unreal Engine の実務経験
- **歓迎スキル**: サーバーサイド開発経験

## デザイナー職

### UIデザイナー

- **仕事内容**: Webサービス・アプリのUI設計
- **必須スキル**: Figma、Photoshop、Illustratorの実務経験

## プランナー職

### ゲームプランナー

- **仕事内容**: ゲームの企画・設計
- **必須スキル**: ゲーム業界での実務経験
