# 面白法人カヤック 採用サイト

「つくる人を増やす」を経営理念に、面白いコンテンツを生み出し続けるクリエイター集団です。

## カヤックについて

面白法人カヤックは、事業内容を「日本的面白コンテンツ事業」としています。1998年創業、2014年東証マザーズ上場。鎌倉に本社を構え、約200名のクリエイターが集う企業です。

### 企業理念：面白法人

1. まずは自分たちが面白がろう
2. 次に周囲からも面白い人と言われよう
3. そして誰かの人生を面白くしよう

### 経営理念：つくる人を増やす

つくる側になることは主体性を持つこと。主体性のある人、いろんなことを自分ごと化できる人を増やしていきます。
