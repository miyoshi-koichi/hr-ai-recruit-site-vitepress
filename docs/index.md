# Kayac Recruit

面白法人カヤックの採用情報サイトへようこそ。

## はじめに

このサイトでは、カヤックの採用に関する情報を提供しています。右下のAIチャットボタンから、いつでも質問できます。

## 主要コンテンツ

### [会社概要](/about)
カヤックの企業情報、理念、事業内容について

### [募集職種](/jobs)
現在募集中のポジション一覧

### [FAQ](/faq)
よくある質問と回答

---

## AIチャットについて

このサイトには、採用に関する質問に答えるAIチャットボットが搭載されています。

- 右下のチャットボタンをクリック
- 採用、業務内容、働き方などについて質問
- 24時間いつでも対応
